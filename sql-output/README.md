# SQL Output Files

This folder contains files generated by the SQL Agents.

| File Pattern | Generated By | Content |
|-------------|-------------|---------|
| `requirements-*.md` | SQL Requirements Analyst | Entities, rules, NFRs, clarifying questions |
| `pseudocode-*.md` | SQL Pseudo Code Developer | Pseudo code, data flow, edge cases |
| `ddl-*.sql` | SQL Code Developer | CREATE TABLE DDL + indexes |
| `code-*.sql` | SQL Code Developer | Stored procedures, queries, views |
| `review-*.md` | SQL Code Reviewer | Findings report with severity levels |
| `tests-*.sql` | SQL Unit Testing Agent | tSQLt test suite |

Files are timestamped: `YYYYMMDD-HHMMSS`
